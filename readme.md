# Notifications Service

## Requirements

- NODEJS [https://nodejs.org/en/](https://nodejs.org/en/)

## Install

```bash
 git clone https://github.com/diovanedm/notification-service-ignite-lab.git
```

```bash
cd notification-service-ignite-lab
npm install
npm run start:dev
```

import the `insomnia-project.json` file to test the application
